# telemetry-cli

Небольшой CLI‑микросервис на Python, который периодически
генерирует тестовую телеметрию и пишет её в таблицу
`telemetry_legacy` в PostgreSQL.

* Период задаётся через `GEN_PERIOD_SEC` (по умолчанию 300 сек).
* Подключение к БД берётся из `DATABASE_URL` либо из стандартных `PG*` переменных.
